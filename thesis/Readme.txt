This is the home folder of the SinoCoin application.

The build folder contains the json files of the smart contracts and can not be modified.
The contracts folder contains the smart contracts.
The data folder contains private data vital for the application.
The migrations folder contains the files that are used to migrate the smart contracts to the Blockchain.
The static folder contains the css style format of the web interface.
The templates folder contains the web application templates.
The test folder contains the test file used by Truffle Suite to check the correctness of the smart contract.

The SinoCoin.py is the Python3 executable of the application.
The truffle-config.js contains the Truffle Suite's configuration data.
