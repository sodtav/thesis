A Web3.0 platform that enables resource-oriented app execution on IoT and Edge devices.

Execute your code remotely and anonymously by paying a small fee, or earn money by registering your device to support the execution!

A better way to execute your code.

Blockchain based

The whole application is based on decentralized methods, such as the Ethereum Blockchain where the transfer transactions take place.

Edge device network

Code is executed on edge devices, providing faster execution times and smaller execution costs.

Passive income

By registering your devices you can earn passive income by executing the code of the users that use the application.

Round robin consensus

Each device enters the round robin mechanism, so each one gets a turn bringing ultimate equity between the users.
